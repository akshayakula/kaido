### splat (scrapped)

Tried preprocessing a handheld video with Meta SAM + LaMa inpainting (via fal.ai)
to remove people before sending to Kiri Engine for 3D Gaussian Splatting.
Pipeline worked end-to-end — 150 frames, ~$1.50 fal cost, ~9 min Kiri build —
producing a `3DGS.ply` splat. Scrapped in favor of the **Franklin** project.
